## Installation

This package can be referenced only
by adding `riot_api` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    #{:riot_api, "~> 0.1.0"}
    {:riot_api, path: "libraries/riot_api"}
  ]
end
```
